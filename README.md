# Lore Keeper

Lore Keeper is an ambitiously detailed project to be a one-stop novel writing software which keeps track of everything related to the Novel's Lore. It's designed for writers to build, organize, and manage every facet of their fictional worlds and stories.

## Features

- **Manuscript Management:** A dedicated module for writing, editing, and organizing your novel's chapters and scenes.
- **Character Management:** Create and manage in-depth character profiles, tracking everything from their backstories, relationships, and attributes to their development arcs throughout the story.
- **Map Creator:** A powerful, grid-based map editor to visualize your world with the following features:
  - Multiple layer support (ocean, grid, objects, elevation, biome, terrain, settlements)
  - A variety of tools, including select, brush, stamp, line, shape, text, and note
  - Biome system with customizable colors
  - Support for both square and hex grids
  - Texture system for natural and man-made elements
- **World Generation:** Procedurally generate worlds with customizable parameters.
- **Project Management:** Organize your world-building projects with a clean and intuitive interface.

## Project Status

This project is currently under active development. The map creator is the most feature-rich module, with ongoing enhancements to the brush and stamp tools, texture system, and layer management. The Manuscript and Character modules are core components and are also being actively developed.
