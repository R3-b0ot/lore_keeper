# Lore Keeper

Lore Keeper is a comprehensive world-building application designed to help you create, organize, and manage your fictional worlds. Whether you're a writer, a game developer, or a hobbyist, Lore Keeper provides the tools you need to bring your imagination to life.

## Features

- **Map Creator:** A powerful, grid-based map editor with the following features:
  - Multiple layer support (ocean, grid, objects, elevation, biome, terrain, settlements)
  - A variety of tools, including select, brush, stamp, line, shape, text, and note
  - Biome system with customizable colors
  - Support for both square and hex grids
  - Texture system for natural and man-made elements
- **World Generation:** Procedurally generate worlds with customizable parameters.
- **Character Management:** Create and manage characters, including their backstories, relationships, and attributes.
- **Project Management:** Organize your world-building projects with a clean and intuitive interface.

## Project Status

This project is currently under active development. The map creator is the most feature-rich module, with ongoing enhancements to the brush and stamp tools, texture system, and layer management. Other modules, such as character management and world generation, are also being actively developed.

See `USER_GUIDE.md` for instructions on how to use the application and `DEVELOPER_GUIDE.md` for information on how to contribute to the project.
