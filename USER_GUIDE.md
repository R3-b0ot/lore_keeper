## Getting Started

To get started with Lore Keeper, simply launch the application and create a new project. You'll be prompted to give your project a name and choose a location to save it. Once you've created a project, you can start building your world.

## The Map Creator

The Map Creator is a powerful tool that allows you to create detailed maps of your fictional worlds. To access the Map Creator, open a project and click on the "Map" tab.

### Layers

The Map Creator uses a layer-based system to help you organize your maps. The following layers are available:

- **Ocean:** The base layer of your map.
- **Grid:** A grid overlay that can be either square or hex.
- **Objects:** A layer for adding objects to your map, such as cities, towns, and dungeons.
- **Elevation:** A layer for creating height maps.
- **Biome:** A layer for defining different biomes, such as forests, mountains, and deserts.
- **Terrain:** A layer for adding natural features, such as rivers, lakes, and coastlines.
- **Settlements:** A layer for adding man-made features, such as roads, bridges, and walls.

### Tools

The Map Creator provides a variety of tools to help you create your maps:

- **Select:** Select and move objects on the map.
- **Brush:** Paint biomes and other features onto the map.
- **Stamp:** Add pre-made objects to the map, such as towns, mountains, and castles.
- **Line:** Draw lines on the map, such as roads and rivers.
- **Shape:** Draw shapes on the map, such as circles and squares.
- **Text:** Add text to the map.
- **Note:** Add notes to the map that are only visible in the editor.

## World Generation

The World Generation module allows you to procedurally generate worlds with customizable parameters. To access the World Generation module, open a project and click on the "World" tab.

## Character Management

The Character Management module allows you to create and manage characters, including their backstories, relationships, and attributes. To access the Character Management module, open a project and click on the "Characters" tab.

## Project Management

The Project Management module allows you to organize your world-building projects with a clean and intuitive interface. To access the Project Management module, open a project and click on the "Project" tab.
